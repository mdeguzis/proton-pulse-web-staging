About this project

Proton Pulse is an open, community-owned compatibility platform for Steam on Linux. Everything -- the data pipeline, hardware reports, voting system, and this site -- lives on GitHub. Fork it. Self-host it. Contribute.

Pulse Reports come from the Decky Loader plugin and capture real hardware specs, Proton versions, launch options, and outcomes from actual gameplay sessions. Reports merge with other open compatibility datasets and ship as static JSON under /data/{appId}/, so they're easy to consume from any tool, script, or frontend. Upvotes and downvotes surface the most helpful configs.

Repositories:
proton-pulse-data -- the data pipeline, static site, and API.
decky-proton-pulse -- the Decky Loader plugin for Steam Deck.
