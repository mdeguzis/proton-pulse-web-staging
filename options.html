Site options

Preferences for this browser, saved locally on your device. They are not tied to a Proton Pulse account and do not sync across devices.

Animations
Background motion: the atom on the home page, the pulse line, and the header. Turning this off cuts CPU and GPU use, which helps on laptops and older hardware. Defaults to off when your system is set to reduce motion.
Default: on (off when your system requests reduced motion).
Default layout
How browse pages lay games out by default. "List" shows horizontal cards stacked top to bottom. "Grid" shows a Steam-style grid of header thumbnails. Each page also has a List/Grid toggle that overrides this for the rest of the session.
Show trend arrow
Small up or down arrow on game cards showing whether Proton compatibility is trending better or worse over the last 90 days compared with the 90 to 270 day window before that. Stable games show nothing. Only appears when there is enough data on both sides of the window to make a fair comparison.
Default: on.
Show library / wishlist icons on cards
Small book (In library) and heart-list (On wishlist) icons that clip onto the store badge in the corner of each game card's artwork. Requires a signed-in Steam account and a synced library / wishlist. Syncs to your account so it follows you across devices.
Default: off.
Store tag size
Size of the store tag (Steam / GOG / Epic) in the corner of each card, including any library / wishlist icon on it. Nudge it up or down until it looks right. Saved on this device.
px
Default: 18.
Show adult games
Include titles flagged by Steam as containing sexual content or nudity in Popular / Recent lists and search. Off by default; toggle on if you want them visible in browse views.
Default: off.
Store badge position
Where to show the store badge (Steam, GOG, Epic) on game cards. "On artwork" overlays the thumbnail corner. "On right" places it alongside the rating pill.
Store badge display
Show the store as a text pill ("Steam", "GOG", "Epic") or a small round icon. Icons take less space, which helps on narrow screens.
Rating layout
Where to show the rating on game cards. "On right" keeps the rating pill in a column next to the title. "Bottom bar" puts the rating into a tier-colored bar across the full bottom of the card -- easier to read on mobile because the title gets the full card width.
Reports per page
How many game cards to preload in each section when browsing reports before the "Load more" button. Higher values show more at once but take a little longer to render.
Home library tiles per page
How many game tiles the home library shows before the "Show more" button. Separate values for phones/tablets and larger screens so each viewport can pick a size that reads naturally. Defaults are 20 on mobile and 50 on desktop.
Auto-load next page
Instead of tapping "Show more" at the bottom of the library, the next page is appended automatically as you scroll near it. Off by default so you keep an explicit stopping point.
Default: off.
Clears all settings on this device and reloads the page.
